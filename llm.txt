# Tray.jl

Tray is an ordered leaf array with a balanced aggregation index in Julia.

## Current Status

The `Tray` module is scaffold-only. Its only current behavior is `Tray.greet()`,
which prints `Hello World!`. Aggregation behavior and APIs are proposals in
OpenSpec and are not implemented.

## Key Concepts

- **Ordered leaves** — source values retain a stable array order
- **Balanced aggregation index** — proposed internal nodes summarize leaf ranges
- **Domain-neutral core** — payload and aggregation semantics are not tied to finance

## Requirements

See [OpenSpec](openspec/) for proposed behavior and the root
[`tray-jl-ears-spec.md`](tray-jl-ears-spec.md) for the combined requirements source.
